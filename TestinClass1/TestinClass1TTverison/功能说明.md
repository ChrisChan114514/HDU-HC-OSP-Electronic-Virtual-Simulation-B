# LCD网格显示与路径寻找系统（课堂测验1-JFT版）

## 硬件配置
- **LCD分辨率**: 800×480
- **网格规格**: 32行 × 40列
- **每个格子**: 20×15 像素
- **LCD时钟**: 33MHz

## 引脚定义
顶层模块 `lcd_rgb_colorbar` 需要以下输入:
- `sys_clk`: 系统时钟
- `sys_rst_n`: 复位信号(低电平有效)
- `switches[3:0]`: 4个开关输入 (B3, B2, B1, B0)

## 功能说明

### 要求1: B0=1 (其他开关=0)
**功能**: 显示基本网格
- 32行 × 40列的网格
- 网格线宽度: 1像素
- 网格线颜色: 黑色
- 背景颜色: 白色

### 要求2: B1=1 (其他开关=0)
**功能**: 在要求1基础上显示目标点和起始点
- **目标点位置**: (30, 8) - 红色显示
- **起始点位置**: (8, 23) - 蓝色显示
- 坐标系: 左上角为(0, 0)

### 要求3: B2=1 (其他开关=0)
**功能**: 在要求2基础上实现目标点闪烁
- 目标点(红色)以2秒周期闪烁
- 起始点(蓝色)保持常亮

### 要求4: B3=1 (其他开关=0)
**功能**: 在要求3基础上实现蓝色方块自动移动
- **起始点**: (8, 23)
- **目标点**: (30, 8)
- **移动速度**: 0.2秒/格
- **移动路径**:
  1. 从起始点水平右移15格 → (23, 23)
  2. 垂直上移15格 → (23, 8)
  3. 水平右移7格到达目标点 → (30, 8)
  4. 返回起始点，循环重复

## 坐标系统
```
(0,0)  (1,0)  ... (39,0)
(0,1)  (1,1)  ... (39,1)
  ...   ...   ...   ...
(0,31) (1,31) ... (39,31)
```

## 颜色定义
- 白色背景: `24'hFFFFFF`
- 黑色网格线: `24'h000000`
- 红色目标点: `24'hFF0000`
- 蓝色起始点/移动块: `24'h0000FF`

## 时序参数 (33MHz时钟)
- 闪烁周期: 2秒 = 66,000,000 时钟周期
- 移动延时: 0.2秒 = 6,600,000 时钟周期

## 使用示例

### 在Quartus II中设置引脚约束
需要在 `.qsf` 文件中添加:
```tcl
# 开关输入引脚 (根据实际FPGA板修改)
set_location_assignment PIN_XX -to switches[0]
set_location_assignment PIN_XX -to switches[1]
set_location_assignment PIN_XX -to switches[2]
set_location_assignment PIN_XX -to switches[3]
```

### 测试步骤
1. **测试基本网格**: 仅打开B0开关
2. **测试点显示**: 仅打开B1开关
3. **测试闪烁**: 仅打开B2开关
4. **测试移动**: 仅打开B3开关

## 注意事项
⚠️ **重要**: 每次只能打开一个开关，确保其他开关为0
